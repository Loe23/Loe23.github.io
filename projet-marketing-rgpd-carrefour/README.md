# Projet : Analyse marketing RGPD-compliant (Carrefour)

## Contexte
Analyse des canaux marketing (Direct, Social, Paid Search, Email) avec respect de la conformité RGPD.

## Stack technique
- BigQuery (entrepôt de données)
- SQL (requêtes d’analyse)
- Looker Studio (dashboard)

## Aperçu du dashboard
![Dashboard Looker Studio](Projet-marketing-rgpd-carrefour)

## Résultats clés
- Canal Direct : 46% du CA (2 100 €)
- Social : 26% (1 200 €)
- Paid Search : 20% (890 €)

## Livrables
- Données pseudonymisées
- Filtrage des consentements RGPD
- Segmentation métier intégrée
